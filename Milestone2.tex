\section{Data Description}
The data is obtained from the Elliptic bitcoin dataset (\url{https://pytorch-geometric.readthedocs.io/en/stable/generated/torch_geometric.datasets.EllipticBitcoinDataset.html}), which contains information about bitcoin transactions. The dataset consists of a graph with over 200K Bitcoin accounts (nodes), 234K directed payment flows (edges), and 166 node features. The nodes features are anonymous due to security concerns. The dataset also contains labels for a subset of the nodes, where $-1$ denotes illicit transactions, $1$ denotes licit transactions, and $0$ denotes unknown transactions.

\section{Refined model sketch}
The model is motivated from the work of \cite{mukherjee2024logistic,daskalakis2020logistic} where they use a logistic regression model with a network regularization term to account for the dependencies between the data points. We will be using a similar approach, but we motivate the model using a bayesian framework. Assume $\sigma_i \in \{-1, 1\}$ is the label of transaction $i$. Let $-1$ denote the illicit class and $1$ denote the licit class. We can model the probability of observing $\sigma_i$ given the features $X_i$ and the graph structure as follows:
\begin{equation}
    P(\sigma_1, \sigma_2, \ldots, \sigma_n | X,G) \propto \prod_{(i,j) \in E} \psi_{X_i, X_j}(\sigma_i, \sigma_j)
\end{equation}
Now, since the labels are binary, we can write the symmetric pairwise potential in a parametric form after taking logs as follows:
\begin{equation}
    \psi_{X_i,X_j}(\sigma_i, \sigma_j) = \exp\left(\beta_{ij} \sigma_i \sigma_j + \alpha_{ij} (\sigma_i + \sigma_j) +c_{ij}\right)
\end{equation}
Here, we make a model assumption that the network effect through each edge is the same, i.e., $\beta_{ij} = \beta$ for all $(i,j) \in E$. Hence, we can write the joint distribution of the labels as follows:
\begin{align*}
    P(\sigma_1, \sigma_2, \ldots, \sigma_n | X,G) & \propto \exp\left(\beta \sum_{(i,j) \in E} \sigma_i \sigma_j + \sum_{(i,j) \in E} \alpha_{ij} (\sigma_i + \sigma_j)\right) \\
                                                  & = \exp\left(\beta \sum_{(i,j) \in E} \sigma_i \sigma_j + 2\sum_{i=1}^n \sigma_i \sum_{j: (i,j) \in E} \alpha_{ij}\right)   \\
\end{align*}
Now, we can model the node-level effect using the covariates as follows:
\begin{equation}
    \sum_{j: (i,j) \in E} \alpha_{ij} = X_i^T \gamma
\end{equation}
Here, $\gamma$ is a parameter vector that captures the effect of the features on the labels. Hence, we can write the joint distribution of the labels as follows:
\begin{equation}
    P(\sigma_1, \sigma_2, \ldots, \sigma_n | X,G) \propto \exp\left(\beta \sum_{(i,j) \in E} \sigma_i \sigma_j + 2\sum_{i=1}^n \sigma_i X_i^T \gamma\right)
\end{equation}
